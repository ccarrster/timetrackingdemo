Cityworks API/Documentation does not appear to be publicly available. There are a few example integrations in github that I found. It appears that some of the requests are available via http GET requests providing JSON in the data parameter.
It looks like Cityworks also offers and SDK which can make integration faster with already writen code.
I found referance to a course, and it looks like the City of Kitchener would already have access to the API via it's existing licence. https://www.cityworks.com/2015/10/new-course-cityworks-2015-apis-programming/
